# МАДОУ д/с №154 — Официальный сайт дошкольного учреждения

Веб-сайт для Муниципального автономного дошкольного образовательного учреждения детский сад №154. Сайт содержит информацию о детском саде, возрастных группах, новостях, контактах и форму обратной связи.

## 📋 Описание

Сайт предоставляет:
- Информацию о детском саде и его деятельности
- Список возрастных групп с описанием
- Новостную ленту учреждения
- Контактную информацию и карту
- Форму обратной связи с сохранением сообщений в базу данных

## 🛠 Технологии

- **PHP** — серверная логика и обработка форм
- **SQLite** — лёгкая встроенная база данных
- **HTML5** — структура страниц
- **CSS3** — стилизация и адаптивный дизайн
- **JavaScript** — мобильное меню, плавная прокрутка
- **Font Awesome** — иконки
- **Google Fonts (Inter)** — современный шрифт

## 🗄 База данных

SQLite база данных создаётся автоматически при первом запуске. Таблица `messages` имеет структуру:

| Поле        | Тип      | Описание                    |
|-------------|----------|-----------------------------|
| id          | INTEGER  | Первичный ключ, автоинкремент |
| name        | TEXT     | Имя отправителя             |
| email       | TEXT     | Email для ответа            |
| message     | TEXT     | Текст сообщения             |
| created_at  | TEXT     | Дата и время отправки       |
