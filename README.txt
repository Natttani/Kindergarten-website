ИНСТРУКЦИЯ ПО ЗАПУСКУ (SQLITE ВЕРСИЯ)

1. Убедитесь, что установлен PHP.
2. Распакуйте проект.
3. Откройте терминал в папке проекта.
4. Выполните команду:

php -S localhost:8000

5. Откройте браузер:
http://localhost:8000

База данных SQLite создастся автоматически при первом запуске.

Технологии:
- HTML
- CSS
- PHP
- SQLite
